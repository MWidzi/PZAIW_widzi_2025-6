# PZAIW_widzi_2025-6
Repozytorium do programowania zaawansowanych aplikacji webowych w roku szkolnym 2025/6

### Legenda folderów:
- projekt na 13-10-2025: first-server
- projekt na 10-11-2025: projekt02 
- projekt na 15-12-2025: projekt03

## README do projekt03
### Konfiguracja: 
Wejść w katalog projekt03 i uruchomić komendę: 
```sh 
npm install
```

### Uruchomienie:
W katalogu projekt03:
```sh 
node index.js
```

Projekt zostanie uruchomiony na [tym adresie](http://localhost:8000)

Link na postronie / prowadzi do "podstrony głównej" aplikacji. 

### Działanie programu:
Klikając w link na niej widoczny zostaniemy przeniesieni do formularza, w ktorym wyświetla się każdy poziom trudnośći każdej piosenki. Używając **checkboxów** zaznaczamy czy mamy na piosence FC czy AP. Po zaznaczeniu tego co chcemy należy kliknąć na przycisk nad tabelą, aplikacja ponownie zabierze nas na "stronę główną", na której tym razem będą widniały zaznaczone piosenki oraz metryka, mająca na celu zwizualizować umiejętności gracza w postaci liczby od 0 do 10.
